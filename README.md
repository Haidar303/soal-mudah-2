Fungsi:
Menentukan apakah bilangan yang dimasukkan adalah ganjil atau genap.

Cara kerja:

Pengguna memasukkan bilangan bulat.

Program memeriksa bilangan % 2.

Jika hasilnya 0 → genap, kalau tidak → ganjil.

Hasil ditampilkan.
